A full-stack web application designed to help students seamlessly track their academic attendance, manage subjects, and strategically monitor their classes. Built with a scalable Node.js backend and a React frontend, this project demonstrates modern web development practices and cloud database integration.

**🚀 Live Demo:** [bunkmanager.onrender.com](https://bunkmanager.onrender.com)

---

## ✨ Key Features
* **Secure User Authentication:** Individualized user accounts with protected routes and personalized dashboards.
* **Cloud-Synced Database:** 24/7 data persistence utilizing a MongoDB Atlas cluster hosted in AWS Mumbai.
* **Dynamic Subject Tracking:** Add, edit, and track up to 15 concurrent subjects with real-time state updates.
* **Data Portability:** One-click CSV export functionality to download attendance records locally.
* **Responsive UI:** Clean, modern interface styled with Tailwind CSS, optimized for both desktop and mobile devices.
* **Continuous Deployment:** Automated CI/CD pipeline linked to GitHub via Render.

---

## 🛠️ Tech Stack
* **Frontend:** React.js (Babel Standalone), Tailwind CSS, HTML5
* **Backend:** Node.js, Express.js REST API
* **Database:** MongoDB Atlas (Mongoose ODM)
* **Hosting / DevOps:** Render (Web Service)
