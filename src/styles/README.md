# Design Tokens

This stylesheet defines CSS variables for the attendance app's design tokens.

- **Color palette**: Adjust `--color-primary`, `--color-accent`, `--color-bg`, etc. to change the theme.
- **Typography**: Modify font families and size variables for headers, body text, and labels.
- **Spacing**: Use the `--space-*` variables for consistent margins and paddings throughout components.

The rest of the application uses these variables in styles or Tailwind overrides; updating them here updates the entire UI.
